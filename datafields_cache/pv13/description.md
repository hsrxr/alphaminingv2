## Dataset description

Category:  *Price Volume*/*Relationship*

Dataset ID: **pv13**

This dataset provides a mapping between Bloomberg IDs (BBID) and Refinitiv Instrument Codes (RIC), enabling users to link securities across Bloomberg and Refinitiv platforms. The data is sourced from both Bloomberg and Refinitiv, and maintained for integration and operational purposes rather than general research. While not directly predictive of price movements, this mapping is essential for harmonizing datasets, ensuring accurate cross-platform security identification, and supporting multi-source data analysis in quantitative research and portfolio management workflows.
